# Exercício: Criando um Componente de Perfil

## Objetivo
O propósito deste exercício é criar um componente de perfil personalizado conforme suas preferências, utilizando os conceitos de componentes e propriedades (props) no React.

## Tarefa
1- Crie um novo componente chamado Profile em um arquivo separado, como Profile.js.<br/>
2- Este componente deve receber as informações do perfil como propriedades (props).<br/>
3- Utilize as props para exibir as informações essenciais, como nome, foto, descrição, etc.<br/>
4- Estilize o componente de acordo com suas preferências pessoais, adicionando CSS conforme necessário.<br/>


## Dicas
* Pense nas informações-chave que você gostaria de incluir em um perfil. <br/>
* Utilize a propriedade props para passar dados dinâmicos para o seu * componente.<br/>
* Experimente adicionar estilos para tornar o perfil visualmente atraente.<br/>
* Lembre-se, a ideia é criar um componente de perfil único que reflete sua abordagem e estilo no desenvolvimento com React. Divirta-se!<br/>
